# 发货单邮件草稿自动生成工具使用说明书

适用对象：业务人员、现场实施人员、系统维护人员  
版本：业务操作培训版  
日期：2026-06-02

## 1. 工具用途

本工具用于自动生成发货单邮件草稿。

业务人员把 Excel 发货单保存到指定客户文件夹后，工具会自动生成一封 Foxmail 草稿邮件，并把 Excel 作为附件放入邮件中。

业务人员仍然需要在 Foxmail 中人工检查草稿，并手动点击发送。

工具不会自动发送邮件。

## 2. 日常业务流程

```text
打开 Foxmail
        ↓
启动发货单邮件草稿工具
        ↓
制作 Excel 发货单
        ↓
保存到客户指定文件夹
        ↓
运行扫描或等待工具自动扫描
        ↓
进入 Foxmail 草稿箱检查邮件
        ↓
确认无误后手动发送
```

## 6. 如何配置客户信息

客户信息在程序界面中配置，不要求业务人员手工修改文件。

操作入口：

```text
打开工具窗口
        ↓
点击“客户配置”页签
```

### 6.1 配置项说明

| 界面字段 | 说明 | 示例 |
| --- | --- | --- |
| 客户编号 | 客户唯一编号，建议使用英文 | `suzhou_fanglin` |
| 客户名称 | 界面和日志中显示的客户名称 | `苏州方林` |
| Excel 文件夹 | 该客户发货单保存位置 | `D:\xusm\桌面` |
| 文件规则 | 哪些 Excel 会被识别 | `*送货单*.xlsx, *送货单*.xls` |
| 扫描子文件夹 | 是否识别 Excel 文件夹下面的子目录 | 勾选 |
| 起始时间 | 只处理修改时间晚于该时间的 Excel | `2026-06-03 09:30:00` |
| 收件人 | 客户收件邮箱，多个邮箱用英文逗号分隔 | `xusm@fanglin.cn` |
| 抄送人 | 抄送邮箱，可为空 | `sales@example.com` |
| 邮件正文 | 生成草稿时使用的正文 | `您好，附件为{subject}，请查收。` |

### 6.2 新增一个客户

操作步骤：

1. 打开工具。
2. 点击 `客户配置` 页签。
3. 点击 `新增客户`。
4. 填写客户编号、客户名称。
5. 点击 `Excel 文件夹` 右侧的 `选择`，选择该客户发货单文件夹。
6. 填写文件规则，例如 `*送货单*.xlsx, *送货单*.xls`。
7. 如果该客户文件夹下面还有月份、订单号等子文件夹，勾选 `扫描子文件夹`。
8. 填写 `起始时间`，或点击 `设为当前时间`。
9. 填写收件人。
10. 如需抄送，填写抄送人；没有抄送可留空。
11. 填写邮件正文。
12. 点击 `保存客户配置`。

### 6.3 Excel 文件夹有子文件夹怎么办

支持。

例如客户文件夹为：

```text
D:\发货单\苏州方林
```

下面还有：

```text
D:\发货单\苏州方林\2026年06月
D:\发货单\苏州方林\2026年06月\第一周
```

只要在客户配置里勾选：

```text
扫描子文件夹
```

工具会下钻扫描客户文件夹下面所有层级的 Excel 发货单。

### 6.4 如何避免历史 Excel 被全部生成草稿

支持设置起始时间。

工具只会处理：

```text
Excel 文件修改时间 > 起始时间
```

例如客户目录中有很多历史 Excel，正式启用时间是：

```text
2026-06-03 09:30:00
```

则在客户配置中填写：

```text
2026-06-03 09:30:00
```

这样早于该时间修改的历史 Excel 不会生成草稿。

如果是在现场首次上线，建议操作：

1. 打开 `客户配置`。
2. 选择客户。
3. 点击 `设为当前时间`。
4. 点击 `保存客户配置`。

这样可以把上线之前的历史 Excel 排除掉。

注意：

```text
起始时间为空时，不按时间过滤，会扫描符合规则的全部 Excel。
```

### 6.5 修改已有客户

操作步骤：

1. 打开 `客户配置` 页签。
2. 在左侧客户列表中选择要修改的客户。
3. 修改 Excel 文件夹、是否扫描子文件夹、起始时间、收件人、抄送人或正文。
4. 点击 `保存客户配置`。

### 6.6 删除客户

操作步骤：

1. 打开 `客户配置` 页签。
2. 在左侧客户列表中选择客户。
3. 点击 `删除客户`。
4. 确认删除。

删除客户配置不会删除已经生成过的 Foxmail 草稿。


## 7. 如何配置 Foxmail 信息

Foxmail 信息在 `系统设置` 页签中配置。

### 7.1 需要重点确认的配置

| 界面字段 | 说明 |
| --- | --- |
| Foxmail 程序 | 选择 `Foxmail.exe` |
| 邮箱本地目录 | 选择 Foxmail 当前邮箱账号的本地存储目录 |
| 电脑重启后自动启动 | 勾选后，下次开机自动打开工具 |
| 自动扫描客户文件夹 | 勾选后，程序会定时扫描客户 Excel 文件夹 |
| 扫描间隔秒数 | 每隔多少秒扫描一次，建议 60 秒 |

### 7.2 Foxmail 路径不一样怎么办

操作步骤：

1. 打开工具。
2. 点击 `系统设置` 页签。
3. 在 `Foxmail 程序` 一行点击 `选择`。
4. 找到并选择 `Foxmail.exe`。
5. 点击 `保存系统设置`。

### 7.3 邮箱账号目录不一样怎么办

操作步骤：

1. 打开工具。
2. 点击 `系统设置` 页签。
3. 在 `邮箱本地目录` 一行点击 `选择`。
4. 选择当前业务邮箱账号的 Foxmail 本地目录。
5. 点击 `保存系统设置`。

常见位置类似：

```text
D:\Program Files\Foxmail\Storage\邮箱账号
```

例如：

```text
D:\Program Files\Foxmail\Storage\xusm@fanglin.cn
```

### 7.4 如何设置开机自动启动

操作步骤：

1. 打开工具。
2. 点击 `系统设置` 页签。
3. 勾选 `电脑重启后自动启动`。

勾选后，工具会写入 Windows 启动项。电脑重启后会自动打开工具窗口，不会弹出 CMD 黑色窗口。

如果不想开机自动启动，取消勾选即可。

### 7.5 点击右上角关闭会不会退出

不会直接退出。

点击窗口右上角 `X` 后，工具只会最小化，程序仍然运行。

如果确实要退出程序，请点击界面上的：

```text
退出程序
```

这样设计是为了避免业务人员误点右上角，导致工具停止工作。

### 7.6 如何开启自动扫描

建议正式使用时开启自动扫描。

操作步骤：

1. 打开工具。
2. 点击 `系统设置` 页签。
3. 勾选 `自动扫描客户文件夹`。
4. 将 `扫描间隔秒数` 设置为 `60`。
5. 点击 `保存系统设置`。

开启后，业务人员将 Excel 保存到客户文件夹后，不需要手动点击扫描。工具会按设置的间隔自动扫描。

说明：

```text
如果上一轮扫描还没有完成，下一轮不会重复启动。
```

建议：

```text
普通业务电脑：60 秒
发货频率很高：30 秒
不建议低于 10 秒
```

## 11. 防重复规则

工具会记录已经处理过的 Excel。

同一个 Excel 文件路径如果已经处理过，再次自动扫描时不会重复生成草稿。

这是为了避免 Foxmail 草稿箱中出现多封重复邮件。

当前默认策略是：

```text
同一路径只自动生成一次草稿。
```

也就是说，即使同一个 Excel 后续又被保存了一次，自动扫描也不会再次生成同主题草稿。

如果确实需要重新生成，请将 Excel 另存为新的文件名，或联系维护人员处理。

## 12. 常见问题

### 12.1 草稿箱没有新邮件

请按顺序检查：

1. Foxmail 是否已经打开。
2. Excel 是否保存完成。
3. Excel 是否放在正确客户文件夹。
4. 文件名是否符合识别规则，例如包含“送货单”。
5. 是否已经生成过相同文件的草稿。
6. 是否已经执行扫描命令。

仍无法解决时，请维护人员查看：

```text
D:\Study\AutoEmail\Code\logs\app.log
```

### 12.2 收件人错误

不要发送。

处理方式：

1. 删除错误草稿。
2. 修改 `customers.json` 中该客户的 `to` 或 `cc`。
3. 重新生成草稿。

### 12.3 主题错误

主题来自 Excel 文件名。

处理方式：

1. 修改 Excel 文件名。
2. 保存文件。
3. 重新生成草稿。

### 12.4 附件错误

不要发送。

处理方式：

1. 删除错误草稿。
2. 检查 Excel 是否放错文件夹。
3. 检查是否保存了错误文件。
4. 重新生成草稿。

### 12.5 重复运行没有生成新草稿

如果文件没有修改，这是正常情况。

如果确实需要重新生成：

1. 修改 Excel 内容后保存。
2. 或者使用新的文件名保存。
3. 再执行扫描。

## 13. 配置修改注意事项

修改配置前建议先备份文件：

```text
D:\Study\AutoEmail\Code\app\config\customers.json
D:\Study\AutoEmail\Code\app\config\settings.json
```

修改时注意：

1. 使用英文双引号。
2. 使用英文逗号。
3. Windows 路径中的 `\` 要写成 `\\`。
4. 不要删除中括号 `[]` 和大括号 `{}`。
5. 修改后先运行一次扫描测试。

如果配置格式错误，程序可能无法启动。
